# 画像トリミング・回転ツール 動作検証ウォークスルー

## 対象
- `/image-crop`
- 画像の切り抜き、90度回転、上下左右反転、PNG/JPEG書き出し

## 手順
1. `/image-crop` を開く。
2. PNG / JPEG / WebP 画像をドラッグ＆ドロップ、クリック選択、またはクリップボード貼り付けで読み込む。
3. 「元画像: 幅×高さpx」が表示されることを確認する。
4. 「左から」「上から」「幅」「高さ」に数値を入力し、プレビューが指定範囲へ更新されることを確認する。
5. 「左90°」「右90°」をクリックし、プレビューの向きと出力サイズが切り替わることを確認する。
6. 「左右反転」「上下反転」を切り替え、プレビューに反転が反映されることを確認する。
7. 「全体に戻す」をクリックし、切り抜き範囲が元画像全体へ戻ることを確認する。
8. 書き出し形式を PNG / JPEG で切り替え、ダウンロードできることを確認する。

## 確認観点
- 画像処理は Canvas API のみで行われ、外部送信は発生しない。
- 範囲外の座標・サイズは画像内へ補正される。
- 90度 / 270度回転時は出力キャンバスの幅と高さが入れ替わる。
- 不正ファイル選択時は日本語エラーメッセージを表示する。
